MATA Tag
==========================================

Tag management for MATA CMS

Installation
------------

- Add the module using composer: 

```json
"mata/mata-tag": "~1.0.0"
```

-  Run migrations
```
php yii migrate/up --migrationPath=@vendor/mata/mata-tag/migrations
```
